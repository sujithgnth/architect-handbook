# TypeScript

Use this section for TypeScript language features and type-system patterns.

Suggested topics:

- type inference and annotations;
- unions, intersections, narrowing, and discriminated unions;
- generics and constraints;
- utility types and mapped types;
- module boundaries and public interfaces;
- error handling and result types;
- React or Angular typing patterns when the lesson is mainly about TypeScript.

Keep examples close to real application code. Add links to official TypeScript
documentation when behavior is version-sensitive.
