and coding patterns that apply beyond one framework.

- values, references, equality, and coercion;
- scope, closures, and modules;
- objects, prototypes, classes, and inheritance;
- async JavaScript: callbacks, promises, `async`/`await`, cancellation;
- collections, iteration, and data transformation;
- errors, debugging, and defensive coding;
- browser APIs when the focus is JavaScript usage rather than frontend product
  design.
